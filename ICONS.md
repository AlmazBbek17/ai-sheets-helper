# Создание иконок для расширения

Вам нужно создать 3 иконки для расширения:
- icon16.png (16x16 пикселей)
- icon48.png (48x48 пикселей)
- icon128.png (128x128 пикселей)

## Вариант 1: Использовать онлайн генератор

1. Откройте https://www.favicon-generator.org/
2. Загрузите любое изображение или создайте простую иконку
3. Скачайте иконки нужных размеров
4. Переименуйте их в icon16.png, icon48.png, icon128.png

## Вариант 2: Создать в Photoshop/GIMP

1. Создайте новый документ нужного размера (16x16, 48x48, 128x128)
2. Используйте синий фон (#1a73e8)
3. Добавьте белый текст "AI" по центру
4. Сохраните как PNG

## Вариант 3: Использовать готовый шаблон

Создайте простую синюю иконку с белыми буквами "AI":
- Фон: синий градиент (#1a73e8 → #174ea6)
- Текст: белый "AI" по центру
- Формат: PNG с прозрачностью

## Быстрое решение

Можете временно использовать любые PNG иконки нужных размеров, чтобы расширение работало. Главное - правильные размеры.
